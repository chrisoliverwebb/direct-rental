# Letting Layer — Brand Guidelines

## Overview

Letting Layer is a modern SaaS platform built for property owners and managers.  
The brand is designed to feel:

- Trustworthy
- Professional
- Long-standing
- Clean and minimal

The visual system prioritises clarity, consistency, and usability.

---

## Logo

### Primary Logo
- Stacked roof layers with chimney
- Represents:
  - Property (roof shape)
  - Platform structure ("Layer")

### Usage Rules
- Always use in primary blue on light backgrounds
- Use white version on dark backgrounds
- Maintain clear space equal to the height of the icon
- Do not:
  - distort
  - rotate
  - change colours
  - add effects (shadows, gradients)

### Icon Only
Used for:
- favicon
- app sidebar
- mobile

## Typography

### Primary Font
**Inter**

Used for:
- all UI
- headings
- body text
- buttons
- labels

### Accent Font
**Playfair Display**

Used for:
- marketing hero text
- key messaging
- emphasis sections

Do not use Playfair in the app UI.

---

### Font Weights

| Weight | Usage |
|-------|------|
| 400 | Body text |
| 600 | UI emphasis, headings |
| 700 | Strong headings |

## Layout Principles

### Visual Ratio
- 90% neutral colours
- 10% primary colour

### Spacing
- Generous whitespace
- Avoid cramped layouts
- Consistent padding system

### Structure
- White cards on light grey backgrounds
- Subtle borders instead of heavy shadows

---

## UI Guidelines

### Buttons

#### Primary
- Blue background
- White text

#### Secondary
- White background
- Grey border
- Dark text

#### Ghost
- No background
- Blue text

---

### Inputs
- White background
- Subtle border
- Blue focus state

---

### Focus State

outline: 2px solid #2563EB;


---

## Iconography

- Style: outline
- Stroke: 2px
- Rounded caps
- Simple, geometric

---

## Brand Tone

The product should feel:

- Calm (not loud)
- Reliable (not trendy)
- Clear (not cluttered)
- Professional (not playful)

---

## What to Avoid

- Too many colours
- Bright/neon tones
- Heavy gradients in UI
- Overuse of accent font
- Inconsistent spacing
- Over-designed components

---

## Summary

Letting Layer is a **modern, trustworthy SaaS brand**.

Design decisions should always prioritise:
1. Clarity
2. Consistency
3. Usability

Over visual flair.
